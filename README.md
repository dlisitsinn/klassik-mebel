# Классик-мебель — сайт мебельного салона

Лендинг для мебельного салона «Классик-мебель» (г. Карталы, Челябинская область): кухни, прихожие и корпусная мебель на заказ.

**Демо:** откройте `index.html` в браузере или разверните через GitHub Pages.

## Возможности

- Адаптивная вёрстка (десктоп / планшет / мобильные, бургер-меню)
- Галерея работ с лайтбоксом (просмотр фото на весь экран, закрытие по Esc)
- Плавное появление секций при прокрутке (IntersectionObserver, с учётом `prefers-reduced-motion`)
- Секции: главный экран, наши работы, преимущества, этапы работы, галерея, контакты
- Кликабельный телефон (`tel:`) и ссылка на сообщество ВКонтакте

## Технологии

- HTML5, CSS3 (Grid, Flexbox, CSS-переменные, `columns` для галереи)
- Ванильный JavaScript, без фреймворков и зависимостей
- Google Fonts: Cormorant Garamond + Manrope
- Фотографии предварительно обработаны (автоконтраст, повышение резкости) в Python/Pillow

## Структура

```
klassik-mebel/
├── index.html      # весь сайт (разметка, стили, скрипты)
├── images/         # фотографии работ (work-01 … work-20)
└── README.md
```

## Публикация на GitHub Pages

1. Создайте репозиторий и загрузите содержимое папки.
2. Settings → Pages → Source: `Deploy from a branch`, ветка `main`, папка `/ (root)`.
3. Сайт будет доступен по адресу `https://dlisitsin.github.io/klassik-mebel/`.

## Если фотографии не загружаются на GitHub Pages

1. Папка `images/` должна лежать **рядом с `index.html`** в корне репозитория (не внутри вложенной папки `klassik-mebel/`).
2. Имена файлов чувствительны к регистру: папка должна называться именно `images`, файлы — `work-01.jpg` … `work-30.jpg`.
3. После загрузки файлов подождите 1–2 минуты и обновите страницу с очисткой кэша (Ctrl+F5).
4. Проверить можно так: откройте `https://dlisitsin.github.io/klassik-mebel/images/work-01.jpg` — если 404, значит папка не загрузилась или лежит не там.
